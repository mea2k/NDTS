## Исходный код

Исходный код содержится в файле [src/index.ts](src/index.ts).

## Запуск программы

Для запуска программы необходимо выполнить команды:
- в рабочем режиме:
   ```
   npm run build
   npm run start
   ``` 
- в режиме отладки:
   ```
   npm run watch
   ```
